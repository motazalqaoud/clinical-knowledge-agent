**Status:** Not started yet — scaffold using AI_PROMPT.md
**Author:** Motaz Alqaoud, PhD — Senior AI/ML Engineer, Biomedical Engineering

## How to build this project

1. Open this folder in VS Code
2. Open GitHub Copilot Chat (or Claude, or Cursor)
3. Copy the entire contents of AI_PROMPT.md into the chat
4. Let the AI assistant scaffold the files, answering its clarifying questions
5. Iterate module by module — review generated code before accepting
